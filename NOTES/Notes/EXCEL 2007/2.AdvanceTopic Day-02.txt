TEXT TO COLUMNS:
======================
GO TO DATA TAB AND THEN SELECT DATA TOOLS-->TEXT COLUMNS
AND SEPARATE UR DATA.

REMOVE DUPLICATES:
===================
GO TO DATA TAB AND THEN SELECT REMOVE DUPLICATES TAB AND THEN DIGLOG BOX WILL OPEN
THERE U SELECT UR CELL AND THEN CLEAR UR DUPLICATES.

DATA VALIDATION:
==================
GO TO DATA TAB AND THEN SELECT DATA VALIDATION
IN THAT WE HAVE TO FOLLOW
1.SETTING
2.INPUT MESSAGE
3.ERROR MESSAGE
